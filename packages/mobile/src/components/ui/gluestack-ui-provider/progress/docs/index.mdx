## API Reference

To use this component in your project, include the following import statement in your file.

```ts
import { Progress, ProgressFilledTrack } from '@/components/ui/progress';
```

```ts
export default () => (
  <Progress>
    <ProgressFilledTrack />
  </Progress>
);
```
